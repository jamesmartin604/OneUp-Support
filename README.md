# OneUp Support

A simple, responsive support page for the OneUp app.

## Overview

This page provides users with a way to contact support via email. It's designed to be hosted on GitHub Pages or any static hosting service.

## Features

- Clean, modern design
- Fully responsive (mobile-friendly)
- No JavaScript required
- Fast loading with system fonts

## Usage

1. Host `support.html` on any static hosting service (GitHub Pages, Netlify, Vercel, etc.)
2. Use the URL as your App Store Support URL
